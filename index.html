Pokecards

Pokecards is a match-type game to help people learn the type matchups in the Pokemon video games. Select a Pokemon's type and select what it has an advantage against to wittle the board down until no cards are left showing. Pokecards was made with React and the Redux store to manage the different states.

TuneBuzz

TuneBuzz is a music discovery application using the Spotify API. Choose three genres from the dropdown provided and TuneBuzz creates a playlist using 5 songs from each of the chosen genres. I worked with a group to create this app. TuneBuzz uses an express server and a Postgres database to store login information and playlists.

TuneChamp

TuneChamp is a music quiz application using the Spotify API. Guess the song out of four options getting only a small clip of the music. Compare your scores with others who have taken the TuneChamp quiz and see if you're the Tune Champion. I worked with a group to create TuneChamp and used React and Redux. It uses a Postgres database to store login information and leaderboard stats.
